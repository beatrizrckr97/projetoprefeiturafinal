Conheça o Projeto Prefeitura mais Próxima de Você

Este projeto está sendo desenvolvido por um grupo de acadêmicos da UDESC, do curso de Tecnologia em Análise e Desenvolvimento de Sistemas. Utilizando dos conhecimentos adquiridos nas disciplinas em decorrer do curso, o projeto está sendo desenvolvido para a disciplina de Projeto Integrador, utilizando dos conhecimentos adquiridos na disciplina de Introdução a Programação Web.

Informações mais detalhadas do projeto se encontram na página inicial.
